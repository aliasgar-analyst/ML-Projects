# Machine Learning Projects

This repository contains a collection of small machine learning projects that I built while learning different ML algorithms using Python.

The goal of these projects was to understand how different models work, how to prepare data for machine learning, and how to evaluate model performance.

Instead of focusing only on theory, I worked on multiple datasets and implemented different algorithms to see how they behave in real scenarios.

## Algorithms Covered

The projects in this repository include implementations of:

- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Naive Bayes
- Random Forest

Each notebook focuses on one algorithm and includes steps such as data cleaning, feature selection, model training, and evaluation.

## Tools & Libraries Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn (for visualization)

## Purpose of This Repository

The purpose of this repository is to document my learning journey in machine learning and build practical experience with commonly used algorithms.

Each notebook explores a different dataset and demonstrates how machine learning models can be applied to solve real data problems.

## What I Learned

Working on these projects helped me understand:

- How different ML algorithms work
- When to use classification vs regression models
- The importance of data preprocessing
- How to evaluate model performance

These projects helped strengthen my foundation in applied machine learning while working with real datasets.
